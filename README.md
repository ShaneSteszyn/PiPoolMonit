PiPoolMonit
==========
Monitor your pool's temperature using a RPI!

## Installation
```
git clone git@github.com:ShaneSteszyn/PiPoolMonit.git
cd poolmonit
npm install
bower install
node config
```

### Server

Use forever or pm2 to keep server going.

    npm install -g pm2
    pm2 start server

or for some dev action:

    node server.js --development
